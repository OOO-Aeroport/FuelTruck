                           Обмен сообщениями с бензовозом


1. Для приема заказа от УНО буду использовать следующий URL:
	/fueltruck/order/номер заказа/объем заказа/номер места самолета

	Расшифровка:
		номер заказа         - Число. Уникальный номер заказа, который генерится в УНО.
		объем заказа         - Число. Количество в тоннах, которое надо залить в  самолет. Генерится в УНО.
		номер места самолета - Число. Номер площадки для самолета, куда надо доехать. Генерится в УНО и должно  быть согласовано с диспетчером движения.

	Пример: /fueltruck/order/125/110/4 - Заказ № 125, Объемом 110 тонн, площадка №4



2. Для запроса массива чекпойнтов до цели буду использовать следующий URL:
	http://ip_сервера:8001/dr/point/текущий чекпойнт/целевое место/номер места

	Расшифровка:
		ip_сервера       - IP адрес сервера диспетчера движения.
		8001             - Порт.
		dr/point         - Обращаемся к диспетчеру движения.
       		текущий чекпойнт - Число. Уникальный номер текущего чекпойнта.
        	целевое место    - Строка. Место назначения. Три типа мест: gas - Заправка, plane - Самолет, garage - Гараж  
        	номер места      - Место. Если самолет, то номер площадки.

	Пример: http://127.5.5.5.9:8001/dr/point/455/gas/3 - Запрос маршрута до заправки №3.
	        http://127.5.5.5.9:8001/dr/point/455/plane/4 - Запрос маршрута до площадки самолета №4.
	        http://127.5.5.5.9:8001/dr/point/455/garage/1 - Запрос маршрута до гаража №1.
		Ожидается ответ в виде JSON с одномерным массивом чисел.



3. Для запроса следующего чекпойнта буду использовать следующий URL:
	http://ip_сервера:8001/dr/point/текущий чекпойнт/целевой чекпойнт

	Расшифровка:
		ip_сервера       - IP адрес сервера диспетчера движения.
		8001             - Порт.
		dr/point         - Обращаемся к диспетчеру движения.
		текущий чекпойнт - Число. Уникальный номер текущего чекпойнта.
		целевой чекпойнт - Число. Уникальный номер целевого чекпойнта.

	Пример: http://127.5.5.5.9:8001/dr/point/455/456 - Запрос на переход от чекпойнта 455 к 456. Ожидается ответ.
        Ожидаемый ответ: OK



4. Для отправки сообщения самолету буду использовать следующий URL:
	http://ip_сервера:8001/fueltruck/plane/номер места самолета/success

	Расшифровка:
		ip_сервера           - IP адрес сервера самолета.
		8001                 - Порт.
		fueltruck/plane      - От кого кому сообщение.
		номер места самолета - Число. Номер места самолета (этот номер я получаю от УНО).
		success              - Успешно заправлено.

	Пример: http://127.5.5.5.4:8001/fueltruck/plane/4/success - Успешно заправлен самолет на площадке 4. Ответа не ожидается.



5. Для отправки сообщения в УНО о завершении миссии буду использовать следующий URL:
	http://ip_сервера:8001/uno/fueltruck/номер заказа/success

	Расшифровка:
		ip_сервера    - IP адрес сервера самолета.
		8001          - Порт.
		uno/fueltruck - Кому от кого сообщение.
 		номер заказа  - Число. Уникальный номер заказа, который приходит из УНО.

	Пример: http://127.5.5.5.23:8001/uno/fueltruck/125/success - Успешно выполнен заказ 125. Ответа не ожидается.
